Скрипт обрабатывает пять блоков данных (data 1–5), извлекая информацию из неструктурированного текста:

Data 1 и Data 2: Основной парсер имен PDF-файлов. Разделяет документы на две группы: с суффиксом «М»/«М.» (где код формируется как редакция_изменение, например 04_03) и обычные файлы без этого суффикса (сохраняется только номер редакции). Вывод построчно адаптирован для прямого копирования в столбец Excel.
Функция split_entries (вспомогательная): Очищает сплошной текст от разделителей (; или перенос строки), превращая его в список чистых значений.
Data 3: Извлекает имя файла и отдельный код вида XX_YY, если они записаны через пробел после расширения .pdf.
Data 4: Находит имена файлов и номера страниц в служебных строках формата «Страница N из Имя.pdf».
Data 5: Собирает готовые полные имена файлов со стандартным суффиксом (RU)_изм_N из раздельных списков названий и кодов, автоматически убирая лишние нули во второй части кода.
