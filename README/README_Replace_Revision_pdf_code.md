Инструкция по работе со скриптом обработки PDF
Скрипт предназначен для автоматизированного редактирования PDF-документов: массовой замены текстовых меток (например, дат) и динамической последовательной нумерации участков (например, XX_X.X → 05_7.1, 05_7.2...).

Основные возможности
Точечная замена: Замена конкретных текстовых строк с полным сохранением стиля (шрифт, размер, цвет).

Последовательная нумерация (режим NUMBER): Автоматический поиск шаблона-заглушки (например, XX_X.X) и замена его на возрастающую последовательность с префиксом.

Откат нумерации (режим RESET): Автоматический поиск всех ранее пронумерованных участков (например, 05_7.1...05_7.100) и сброс их обратно к шаблону XX_X.X. Это позволяет редактировать документ и запускать перенумерацию повторно без ручной очистки.

Алгоритм работы
Анализ стиля: Скрипт определяет параметры (размер шрифта, цвет) исходного текста, чтобы новая надпись визуально соответствовала документу.

Безопасность: Используется инструмент add_redact_annot (закрашивание белым) для удаления старого текста, что гарантирует чистоту документа без "двоения" букв.

Гибкость: Скрипт автоматически обрабатывает повернутые страницы и использует матрицы преобразования координат.

Настройка режимов (блок if __name__ == "__main__")
Переключайте переменную MODE для выбора задачи:

1. Режим NUMBER (Нумерация)
SEQUENCE_SEARCH: Шаблон, который ищем (обычно XX_X.X).

SEQUENCE_PREFIX: Префикс для новых номеров (например, 05_7).

SEQUENCE_START: Начальный номер счетчика.

REPLACEMENTS: Дополнительный список точечных замен (дата, ФИО и т.д.).

2. Режим RESET (Сброс к шаблону)
RESET_PATTERN_PREFIX: Префикс, который нужно найти для сброса (например, 05_7). Скрипт найдет все значения вида 05_7.1, 05_7.2 и т.д.

RESET_TEMPLATE: Значение, на которое будут заменены найденные номера (обычно XX_X.X).

Структура функций
replace_text_in_pdf: Основная функция для точечных замен.

replace_sequence_in_pdf: Функция для выполнения нумерации по шаблону.

reset_to_template: Функция для автоматического сброса нумерации с использованием регулярных выражений (re).

find_font_info: Вспомогательная функция для извлечения параметров шрифта из блоков PDF.

find_pdfs: Утилита для фильтрации файлов (исключает из обработки уже созданные скриптом файлы по суффиксу SUFFIX).

Рекомендации
Всегда указывайте путь к шрифту GOST2304A.ttf, чтобы сохранить соответствие ГОСТу.

При добавлении новых участков: сначала запустите MODE = "RESET" с соответствующим префиксом, внесите изменения в PDF, затем запустите MODE = "NUMBER" для финальной перенумерации.
