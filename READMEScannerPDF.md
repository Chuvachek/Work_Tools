Другие способы "сканировать" PDF

qpdf (CLI) — декомпилирует PDF в читаемый текстовый QDF-формат, удобно смотреть структуру объектов глазами: qpdf --qdf --object-streams=disable in.pdf out.qdf
pikepdf (Python, обёртка над qpdf) — альтернатива PyMuPDF, иногда удобнее для низкоуровневой работы со словарями объектов.
pdfplumber — хорош для анализа таблиц/расположения текста по координатам.
mutool (входит в MuPDF) — CLI-утилита, mutool show file.pdf trailer / mutool clean -d для просмотра сырой структуры без Python.
Просто текстовый редактор — открыть PDF как текст (для файлов без сжатия потоков) — быстро понять на глаз структуру, но большинство современных PDF используют сжатие (FlateDecode), поэтому напрямую не читается.
